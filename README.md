# Drupal-blog-website
This is a sample druapl blogging site clone.

Enviournment:

   1) Colima
   2) Composer
   3) Ddev
   4) Drupal-9
   5) Terminal 
   
Project setup steps:

   1) Intall Colima and run command: colima start
   2) cd to project folder
   3) Run command: ddev start
